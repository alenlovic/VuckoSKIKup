O nama


Web aplikacija "Vučko SKI Kup" je osmišljena da radi na principu upravljanja informacija o prijavljenim takmičarima i rasporedu skijaških utrka.


Razvoj aplikacije je rađen pomoću ASP.NET tehnologije sa Entity frameworkom, te HTML, CSS i Javascriptom na Frontendu. Za bazu podataka je korišten Microsoft SQL Server.
